Welcome to Birthday Tracker App

Your personal way to keep track of birthdays and special occasions.

Project Overview

For my MERN stack final project, I will be building a Birthday Tracker App from scratch. The application will allow users to register, log in, and manage a personalized list of friends and family members’ birthdays. Users will be able to create and categorize contacts into groups (e.g., "Family", "Work", "Friends"), view upcoming birthdays, and receive reminders.

Why This Project?

This is a real-world app I would personally use. It helps keep track of important dates, avoids forgetting birthdays, and makes it easy to organize loved ones into meaningful categories.

Technologies to be Used:

  • Frontend: React + Tailwind CSS (or Bootstrap)
  • Backend: Node.js with Express
  • Database: MongoDB Atlas
  • Authentication: JWT-based login (possibly using Auth0 or a custom solution)
  • Hosting:
    • Frontend: Vercel or Netlify
    • Backend: Render or Railway
    • Database: MongoDB Atlas (Cloud)

Core Features:

  • User registration & login (JWT authentication)
  • Add, edit, and delete birthday entries
  • Categorize contacts into custom groups
  • View upcoming birthdays (sorted by date)
  • Visual reminders for upcoming birthdays
  • Many-to-many relationship:
    • One user can have many contacts
    • A contact can belong to many groups

Sample Data:

I will generate and insert at least 12 sample contacts across 3–4 groups, with realistic names, dates, and categories (via AI or mock data tools like Mockaroo).
